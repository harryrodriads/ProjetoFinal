📜 Política de Privacidade

Última atualização: Março de 2025

Nos comprometemos a proteger seus dados pessoais. Esta página explica como suas informações são coletadas, utilizadas e protegidas, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018).

📌 1. Coleta de Dados

Ao utilizar o SGHSS, podemos coletar os seguintes dados pessoais:

  • 📌 Dados cadastrais: Nome, e-mail, unidade de trabalho.
  • 📌 Dados de acesso: Endereço IP, navegador e dispositivo utilizado.
  • 📌 Cookies: Armazena preferências de navegação para melhorar sua experiência.

🎯 2. Finalidade do Uso

Os dados são utilizados exclusivamente para:

  • ✅ Autenticação e segurança do usuário.
  • ✅ Gestão hospitalar e administrativa.
  • ✅ Melhoria contínua da plataforma.

⚖️ 3. Direitos do Usuário

Você possui os seguintes direitos conforme a LGPD:

Direito Descrição
📄 Acesso Você pode solicitar uma cópia dos seus dados pessoais armazenados.
✏️ Correção Seus dados podem ser atualizados ou corrigidos mediante solicitação.
🗑️ Exclusão Você pode solicitar a remoção dos seus dados do sistema.
🔒 Revogação de Consentimento Você pode retirar seu consentimento para uso dos seus dados a qualquer momento.

🔒 4. Segurança dos Dados

Utilizamos medidas de segurança para proteger seus dados, incluindo:

  • 🔐 Criptografia de senhas e dados sensíveis.
  • 📑 Monitoramento contínuo para detectar atividades suspeitas.
  • 🛡️ Controle de acesso restrito apenas a usuários autorizados.
